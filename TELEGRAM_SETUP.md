# Отзывы и заявки — Google Apps Script

См. **`google-apps-script/README.md`** — полная инструкция по настройке.

## Кратко

1. Google Таблица + Apps Script (`google-apps-script/Code.gs`)
2. Deploy → Web App → скопировать URL
3. Render: `GAS_WEB_APP_URL` + `GAS_SECRET`
4. Отзывы: форма на сайте → письмо с **Принять / Отклонить** → на сайте
5. Заявки: форма на сайте → письмо + Telegram бот

Firebase **больше не используется**.
