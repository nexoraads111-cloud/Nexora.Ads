# Google Apps Script — отзывы и заявки на почту

Вместо Firebase: **Google Таблица** + **Gmail** + кнопки **Принять / Отклонить**.

## Быстрая настройка (5 минут)

### 1. Создайте таблицу
1. Откройте [Google Drive](https://drive.google.com) → **Создать** → **Google Таблицы**
2. Назовите: `NexoraWeb Data`

### 2. Вставьте скрипт
1. В таблице: **Расширения** → **Apps Script**
2. Удалите код по умолчанию
3. Скопируйте весь файл `Code.gs` из этой папки
4. В `CONFIG.ADMIN_EMAIL` укажите **вашу почту Gmail**
5. Сохраните (Ctrl+S)

### 3. Первый запуск
1. Выберите функцию `setup` → **Выполнить**
2. Разрешите доступ к таблице и почте
3. В таблице появятся листы **Reviews** и **Orders**

### 4. Опубликуйте Web App
1. **Развернуть** → **Новое развертывание**
2. Тип: **Веб-приложение**
3. Выполнять как: **Я**
4. Доступ: **Все**
5. Скопируйте **URL веб-приложения**

Пример URL:
```
https://script.google.com/macros/s/AKfycbx.../exec
```

### 5. Добавьте URL на Render
В [Render Dashboard](https://dashboard.render.com) → сервис API → **Environment**:

| Переменная | Значение |
|------------|----------|
| `GAS_WEB_APP_URL` | URL из шага 4 |
| `GAS_SECRET` | тот же секрет что в `CONFIG.SECRET` в Code.gs |

Удалите (не нужно):
- `FIREBASE_DATABASE_URL`

### 6. Перезапустите Render
После сохранения переменных Render перезапустит API.

---

## Как работает

### Отзывы
1. Клиент заполняет форму на сайте → **Отправить**
2. Вам на почту письмо с кнопками **✅ Принять** / **❌ Отклонить**
3. **Принять** → отзыв появляется на сайте в разделе «Отзывы»
4. **Отклонить** → не показывается

### Заявки
1. Клиент отправляет заявку на сайте
2. Вам на почту письмо с данными

---

## Проверка

Откройте в браузере:
```
https://ВАШ-GAS-URL/exec?action=reviews
```
Должен вернуться JSON: `[]` или список отзывов.

---

## Безопасность

- `GAS_SECRET` защищает отправку отзывов/заявок через API
- Кнопки Принять/Отклонить защищены уникальным `token` в каждом письме
