Сделайте эти шаги **один раз** на почте `nexoraads111@gmail.com`.

## Шаг 1 — Откройте скрипт

1. Перейдите: https://script.google.com
2. Найдите проект с URL `AKfycbxvE3e_uv8rYURscjC3YJxVmMmTLppuQvpQOzIRsRUI-Ngd5_88gniscbB-P4JlRfs4`
   - Или создайте **Новый проект** и вставьте код из `Code.gs`

## Шаг 2 — Вставьте новый код

1. Удалите весь старый код
2. Скопируйте **весь** файл `google-apps-script/Code.gs` из репозитория
3. Вставьте в редактор Apps Script
4. Сохраните (Ctrl+S)

## Шаг 3 — Запустите setup()

1. Вверху выберите функцию **`setup`**
2. Нажмите **Выполнить** ▶
3. Разрешите доступ:
   - Google Drive (создать таблицу)
   - Gmail (отправка писем)
   - Google Таблицы
4. В логе должно появиться: `OK — таблица: https://docs.google.com/...`

## Шаг 4 — Опубликуйте заново (обязательно!)

1. **Развернуть** → **Управление развертываниями**
2. Нажмите ✏️ на текущем развертывании
3. **Версия** → **Новая версия**
4. Тип: **Веб-приложение**
5. Выполнять как: **Я**
6. Доступ: **Все**
7. **Развернуть**

URL останется тот же:
```
https://script.google.com/macros/s/AKfycbxvE3e_uv8rYURscjC3YJxVmMmTLppuQvpQOzIRsRUI-Ngd5_88gniscbB-P4JlRfs4/exec
```

## Если письма не приходят

- Проверьте папку **Спам**
- Убедитесь, что `setup()` выполнен без ошибок
- В GAS: **Выполнения** — смотрите ошибки после отправки формы
